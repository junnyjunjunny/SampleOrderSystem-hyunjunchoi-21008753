## 사용 방식

각 메뉴 기능을 개발/수정할 때는:
1. 해당 번호의 문서를 먼저 읽고, 이번에 바뀌는 요구사항에 맞춰 문서를 갱신(토론 후)한다.
2. 문서 갱신 내용을 바탕으로 [PLAN_TEMPLATE.md](../PLAN_TEMPLATE.md)를 복사해 `Plan.md`를 작성한다.
3. [CLAUDE.md](../CLAUDE.md)의 "작업 진행 방식"(Plan.md/docs 갱신 → 구현)으로 진행한다. 엄격한 RED-GREEN 선행 확인은 이 프로젝트에서 완화되어 있다([CLAUDE.md](../CLAUDE.md) 참고).

전체 도메인/아키텍처는 [PRD.md](../PRD.md) 참고.
